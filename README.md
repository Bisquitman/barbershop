# gulp-2022 на основе сборки от Фрилансера по жизни

## Project setup

```
npm i
```

## Возможности сборки

### Режим разработки:
```
npm run dev
```

### Сборка проекта:
```
npm run build
```

### Сборка svgsprite:
```
npm run svgsprite
```

### Создание архива проекта:
```
npm run zip
```

### Залить проект на ftp:
```
npm run deploy
```
